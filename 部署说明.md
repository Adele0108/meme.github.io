# 表情包生成器 - 部署说明

## 📱 两种使用方式

### 方式一：部署为在线网站（推荐）✨

部署后可以：
- ✅ 获得一个公开的网站URL
- ✅ 生成二维码供他人扫码访问
- ✅ 随时随地使用，无需安装
- ✅ 多人共享使用

### 方式二：本地运行

适合：
- 🔒 个人使用，不需要分享
- 🚀 快速测试和开发
- 📦 离线使用

---

## 🌐 部署为在线网站

### 方案1：GitHub Pages（免费，推荐）⭐

**步骤：**

1. **创建 GitHub 仓库**
   - 访问 https://github.com
   - 点击右上角 "+" → "New repository"
   - 仓库名：`meme-generator-v2`（或任意名称）
   - 选择 Public（公开）
   - 点击 "Create repository"

2. **上传代码**
   ```bash
   # 在 Meme-generator-v2 文件夹中打开命令行
   git init
   git add .
   git commit -m "Initial commit"
   git branch -M main
   git remote add origin https://github.com/你的用户名/meme-generator-v2.git
   git push -u origin main
   ```
   
   或者直接在 GitHub 网页上点击 "uploading an existing file"，拖拽所有文件上传

3. **启用 GitHub Pages**
   - 进入仓库页面
   - 点击 Settings（设置）
   - 左侧菜单选择 Pages
   - Source 选择 "Deploy from a branch"
   - Branch 选择 "main" / 文件夹选择 "/ (root)"
   - 点击 Save

4. **访问网站**
   - 等待 1-2 分钟后，访问：`https://你的用户名.github.io/meme-generator-v2/meme-generator.html`
   - 或者：`https://你的用户名.github.io/meme-generator-v2/`（如果设置 index.html）

5. **生成二维码**
   - 使用在线工具：https://cli.im/
   - 或使用微信小程序"二维码生成器"
   - 输入网站URL，生成二维码
   - 其他人扫码即可访问

---

### 方案2：Netlify（免费，最简单）🚀

**步骤：**

1. **访问 Netlify**
   - 打开 https://www.netlify.com
   - 使用 GitHub 账号登录（推荐）

2. **部署**
   - 点击 "Add new site" → "Deploy manually"
   - 或者拖拽整个 `Meme-generator-v2` 文件夹到页面
   - 等待部署完成

3. **获得URL**
   - 部署完成后会得到一个类似 `xxx.netlify.app` 的URL
   - 可直接访问，也可以自定义域名

4. **生成二维码**
   - 使用URL生成二维码分享给他人

---

### 方案3：Vercel（免费，适合开发）⚡

**步骤：**

1. **访问 Vercel**
   - 打开 https://vercel.com
   - 使用 GitHub 账号登录

2. **导入项目**
   - 点击 "Add New Project"
   - 选择你的 GitHub 仓库
   - 或直接上传文件夹

3. **部署**
   - Framework Preset 选择 "Other"
   - 点击 Deploy
   - 等待部署完成

---

### 方案4：腾讯云/阿里云/其他云服务

如果有云服务器或对象存储服务，可以：
- 上传到云存储的静态网站托管
- 配置 CDN 加速
- 绑定自定义域名

## 📦 本地运行（测试用）

如果只是个人使用，可以参考 `运行说明.txt`：

1. **使用 VS Code Live Server**
   - 安装 Live Server 扩展
   - 右键 HTML 文件 → Open with Live Server

2. **使用 Python**
   ```bash
   python -m http.server 8000
   ```
   访问：http://localhost:8000/meme-generator.html

3. **使用 Node.js**
   ```bash
   npx http-server
   ```

## 🔒 注意事项

1. **API Key 安全**
   - 通义 API Key 会暴露在前端代码中
   - 建议：如果部署公开网站，提醒用户填写自己的 API Key
   - 或者创建后端代理（更安全）

2. **CORS 问题**
   - 如果遇到跨域问题，可能需要配置代理
   - 使用 Netlify/Vercel 可以配置 `_redirects` 文件

3. **文件大小**
   - 确保 `images` 文件夹中的图片不要太大
   - 建议压缩图片以加快加载速度
